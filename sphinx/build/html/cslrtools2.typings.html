class cslrtools2.typings.SupportsArray(*args, **kwargs)[source]

Bases: Protocol, Generic

Protocol for objects that support conversion to arrays.

This protocol defines objects that implement the __array__() method, allowing them to be converted to NumPy arrays or similar array types.

Type Parameters:

R: The return type of __array__(), must be array-like.

Example

Custom class implementing the array protocol:

>>> import numpy as np
>>> class MyArray:
...     def __init__(self, data):
...         self.data = data
...     def __array__(self, dtype=None, copy=None):
...         return np.array(self.data, dtype=dtype)
>>> obj = MyArray([1, 2, 3])
>>> np.asarray(obj)
array([1, 2, 3])
